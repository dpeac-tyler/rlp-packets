Additional Item

Instructions:

Enter the name of the Additional Item. You can add more items by clicking the Add Additional Item button.

For each item, choose the Minimum and Maximum Quantity that the user can purchase.

Set the Minimum value to zero (0) if the item is an optional purchase.

Set the Minimum value to one (1) if the user is required to buy at least one.

Set the Maximum value to any number between one (1) and nine hundred ninety-nine (999) as appropriate.

Each Additional Item can have different fee amounts for different groups or categories (e.g. Child, Senior, Veteran). You must enter at least one Price Point Fee Name and Price amount. You can add price points by clicking Add Price Point under that same group or category. If you need to add another group, then more can be added by clicking Add Additional Item.
