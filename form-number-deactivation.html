Form Number, Deactivation Date & Search Terms

For this step, you can provide an application a number. In most situations, this is the same number as that used on a paper form. This field is optional. You can also continue building the application and add an application number later. Once the application goes live, you can no longer edit this number.

"Application Deactivation Date" is the last date that you want this application to be available online. This date must be greater than or equal to the current date and the application go-live date.

Type the Form Number in the field below.

Application Deactivation Date

Type in the search terms that users can use to search for this application on the internet.
