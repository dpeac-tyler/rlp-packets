Application Title, Display Name, Subtitle, Descriptions & URL

Now let's give your application a title, display name, subtitle, description and URL.

The application title is important because it is saved as a unique name for your application. The display title will appear at the top of every page of the application and it can be changed at anytime without affecting the saved application title. The display title will also appear on the Certificate the user receives. You also have the option to give your application a subtitle.

The description should be a brief explanation of the application to give users a clear understanding of the application they are about to submit.

NOTE: The More Information URL is not the URL that will be assigned to this particular application. Instead, it is a link to a website that may have more comprehensive information, FAQs, etc. for users that need additional details pertaining to the license/permit application they are submitting.
