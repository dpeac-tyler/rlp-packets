Task Management

In the Task Management section, define the tasks and which roles(s) have permissions to review them as part of the submission review process.

- Delete Task

Task List

You may add up to five (5) task(s) to your review process. The order in which the task(s) appear below will be reflected in the review process.

For each task, select the User Group that will be assigned to that task. The assigned user group will have the following permissions on the selectable task: Claim, Approve, Reject, Deny, and Unclaim.

If you need to review user groups within your organization, go to settings | Users/Groups from the Home page prior to setting permissions for this application.

Standard - Allows you to use the standard task names as your task naming convention, Task x of y, e.g. for a two-step process Task 1 of 2, Task 2 of 2.

Agency Defined - Allows you to select task names from those defined by your agency.
